# Telegram Weather Bot

Простой Telegram-бот на Python, который показывает погоду в любом городе.

### Функции
- Кнопки меню
- Погода в Алматы одной кнопкой
- Погода по названию города (поддержка русского и английского)
- Использует OpenWeatherMap API

### Как запустить
1. Установите библиотеку: `pip install pyTelegramBotAPI requests`
2. Получите API-ключ на https://openweathermap.org/api
3. Вставьте свой токен бота и API-ключ в код
4. Запустите: `python bot.py`
